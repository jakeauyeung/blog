###安装

前提先要安装好homebrew

```
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)”
```
之后安装weechat

```
brew install weechat
```

会花几分钟安装相关依赖。

运行weechat

```
weechat
```

添加IRC服务器

```
weechat> /server add example irc.example.com/6669 -ssl -password=PASSWORD -ssl_verify=no
weechat> /connect example
```
加入IRC频道

```
/join #atom
```
